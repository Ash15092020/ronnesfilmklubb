Slumpmässig Skog

Idag, torsdag (2025-11-27), så byggde vi vår andra modell. Modellen var gjord för att visa hur elever hade presterat på ett test innan och ett test efter, utifrån några faktorer, antal timmar pluggade, tidigare resultat, Hours Studied', 'Previous Scores', 'Extracurricular Activities', 'Sleep Hours', 'Sample Question Papers Practiced. Detta visade vi genom att ett horisontellt stapeldiagram av den slumpmässiga skogen

Detta gjordes med python inne på google colabratory. Där vi först började med att importera några färdiga paket och sedan importerade vi själva datan som vi skulle ha. Efter det så läste vi in den datan/filen som vi importerade med hjälp av paketet pandas. Sedan processerade vi datan, och med hjälp av One Hot Encoding så gjorde vi om en av faktorernas kolumn till vektorer, eftersom att det var den enda kolumnen som var icke numrerisk. Därefter omvandlades de vektorerna till dummyvariabler.

Efter att vi hade arbetat med datan så var det dags att välja och initera vilken modell vi skulle ha, vilket var en slumpmässig skog. När vi väl hade valt modell så tränade vi modellen med en väldigt stor datamängd på, därefter att det var tränat så testade vi modellen med en annan datamängd. Innan vi kunde testa modellen med datamängden var vi tvungna att ladda upp den filen och bearbeta med datan på samma sätt som tidigare. När vi väl hade testat modellen så jämförde vi med det vi hade fått fram ifrån den nya datamängden och de faktiska värdena, vilket visade sig stämma hyfsat bra överens.

Till sist så var det dags att rita upp själva stapeldiagramet över den slumpmässiga skogen med hjälp av paketet "matplotlib.pyplot".
